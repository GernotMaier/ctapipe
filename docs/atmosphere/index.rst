=====================
Atmosphere module (`atmosphere`)
=====================

.. currentmodule:: ctapipe.atmosphere

Introduction
============

`ctapipe.atmosphere` contains functions and classes to handle atmosphere related stuff

Currently this includes handling atmospheric density and transmission profiles, and a simple reduction of elastic Lidar data
